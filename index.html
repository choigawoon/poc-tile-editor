🧩 POC Tile Editor

A tile pipeline split into three decoupled roles, joined only by a bundle and a shared @poc/core SDK. Vite + PixiJS monorepo.

role ①

Editor — the tool

Paint, fill, rect, layers; autosave; ▶Play the live map. Exports to Godot / Unity / Tiled / Generic.

Open editor → (npm run dev → :5173)
→
role ②

Bundle — the contract

Data + resources only, no logic: map.json (tile-id grid) + tileset PNG(s). The seam between tool and game.

bundles/demo/
→
role ③

Game — the runtime

PixiJS/WebGL. Consumes a bundle and nothing else. Owns collision, player, camera, skins. Press T for day/night.

Open demo game → (npm run dev:game → :5175)

Directory map

poc-tile-editor/
├── packages/core/   🔑 @poc/core    shared SDK: gid math + exporters (no DOM)
├── apps/editor/     ① the tool      Vite app · ▶Play · autosave
├── bundles/         ② the contract  demo/ → map.json + tileset.day/night.png
├── apps/game/       ③ the game      Vite + PixiJS · consumes a bundle
│
├── engine-templates/   Godot (.gd) & Unity (.cs) runtime importers
├── tools/              make-demo-bundle · make-tileset-variant · sync-dist-bundle
├── samples/            source tilesets
├── ARCHITECTURE.md     the three roles, the bundle contract, reskin rules
└── README.md

Dev (HMR): npm install, then npm run dev (editor :5173) and npm run dev:game (game :5175). In the editor, hit ▶ Play to run the live map. Release: npm run build:all. Rebuild the demo bundle: npm run bundle:demo.
