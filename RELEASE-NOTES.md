# 릴리스 노트 — POC Tile Editor

> 무엇이 언제 추가됐는지 한눈에. 최신이 위. (에디터 **Help → 릴리스 노트**에서도 열림)
>
> **기록 규칙**: 새 작업은 `## YYYY-MM-DD` 날짜 섹션 아래 영역별 묶음으로 적는다. 한 줄은 *사용자가 체감하는 변화* 위주(내부 리팩터는 괄호로). 같은 날 여러 건은 같은 날짜 섹션에 모은다.

## 요약
타일맵 에디터가 단순 그리기 툴에서 **워크스페이스 기반 저작 환경**으로 성장: 멀티맵 탭 · 타일 메타/계층태그 · 공유 SDK · 패턴 & PCG 던전(수동·자동) · AI 타일셋 생성 · 도킹 패널 + 메뉴바 · 인앱 문서.

## 2026-05-30

### UI 셸 — 도킹 패널 + 메뉴바
- **도킹 패널**: 헤더 드래그로 좌/우 컬럼 도킹(드롭존 하이라이트 + 삽입선) 또는 플로팅(드래그·리사이즈). ✕ 닫기 · ▾ 접기 · ⤢ 플로팅. 컬럼 밖에 놓으면 **원위치 복귀**(분실 방지). 배치·숨김 영속.
- **메뉴바**: File · Edit · View(패널 표시/숨김 + Reset layout) · Tools · Help. ▶ Play는 우측 버튼. (닫힌 패널은 View에서 다시 켬)
- **트랙패드 입력**: 두 손가락 스크롤=이동, 핀치/⌘·Ctrl+스크롤=줌. (스페이스/중클릭 드래그 이동도)

### PCG 던전
- **Dungeon (PCG) 도킹 패널** — Cols/Rows/Room/Seed 인라인 입력(팝업 연속입력 제거).
- **Auto-build** — 패턴 없이 방을 절차 생성(바닥+벽, 문 자리 틈)해 미로에 조립, **연결 보장**. 바닥/벽 타일은 태그·solid로 자동 선택. 시드 주면 결정적.
- **From my patterns** — 내가 만든 패턴을 문 매칭으로 조립.

### AI 타일셋
- 패널 입력을 **Cols×Rows×Tile ↔ 해상도 연동**으로 정리(슬라이스 격자 정확) + 미리보기 격자 오버레이.
- 생성물이 **추가 즉시 팔레트에 선택**돼 바로 사용.

### 문서
- **인앱 문서 뷰어**(Help 메뉴): 역기획서·릴리스 노트를 **전체화면**으로 열람, ✕/Esc로 에디터 복귀.
- 역기획서를 두괄식 + 점진적 공개(목차 + 접기)로 재작성. 릴리스 노트 신설.

## 2026-05-29

### 기반 (모노레포)
- 탑다운 타일 에디터(브러시·레이어·undo·autosave·▶Play) + PixiJS 게임 + 번들 계약. 3역할 분리.
- **Turborepo** 전환(의존성 그래프 + 캐시).

### 타일 메타데이터 + 계층 태그
- 타입 메타(`solid`·임의 props) + 인스턴스 객체 + **Unreal식 계층 태그**(`terrain.water.deep`).
- "Selected tile" 패널: Solid · **태그 트리**(추가/이름변경 캐스케이드/삭제) · 프로퍼티. 팔레트에 메타 카테고리 구슬(🔴/🔵/🟢). 태그/키 소문자 저장.

### 공유 SDK (`@poc/core`)
- `gid`·`meta`·`tags`·`query`·`exporters`. 게임 충돌·태그 질의를 한 정의로 공유(`cellTags`/`gidHasTag`/`findCells` 등).

### 멀티 맵 + 워크스페이스
- **멀티 맵 탭**(전환·새 맵·이름변경·닫기, 맵 간 격리, 타일셋 공유).
- (내부) 공유 워크스페이스 모델 + `state.project` Proxy로 무수정 전환, 구버전 저장본 자동 마이그레이션.

### 패턴
- 패턴 = 맵형 작은 문서 + 엣지 문(N/E/S/W). 탭바 **Maps | Patterns** 분리.
- **스탬프 툴(▦)**: 패턴을 맵에 덩어리로 찍기(레이어명 매칭·빈셀 스킵·1 undo).

### AI 타일셋 (genai MCP)
- 에디터 안 **"✨ AI tileset" 패널** + Node **dev 브리지**(`/api/genai/generate`, MCP 핸드셰이크). 프리셋 프롬프트.

### 도구·Export
- 이미지→타일 임포트(픽셀해시 dedupe). 라벨 테스트 타일셋 생성기. Export 4종(Generic/Tiled/Godot4/Unity).
