## ✅ 지금 상태: 전부 동작 (5단계 완료)

```
[에디터로 그림] → [번들로 추출] → [게임이 로드해 플레이]   ← 전 구간 연결됨
     ①                  ②                   ③
```

| | 역할 | 폴더 | 스택 | 핵심 |
|---|---|---|---|---|
| ① | **에디터** (툴) | `apps/editor/` | Vite | 페인트·레이어·undo·autosave·▶Play·Export |
| ② | **번들** (계약) | `bundles/` | JSON+PNG | `map.json`(타일 id 격자)+타일셋, **로직 없음** |
| ③ | **게임** (런타임) | `apps/game/` | Vite+**PixiJS** | 번들만 소비·충돌·플레이어·낮밤스킨 |
| 🔑 | **코어** (공유) | `packages/core/` | 순수 JS | gid 계산+exporters, 에디터·게임 공용 |

## 🚀 실행 (외우기 쉽게)

| 명령 | 결과 |
|---|---|
| `npm install` | 1회 설치 |
| `npm run dev` | 에디터 → :5173 |
| `npm run dev:game` | 게임 → :5175 |
| `npm run build:all` | 출시 빌드 (editor/dist + game/dist) |

조작: 에디터에서 **▶ Play** 누르면 즉시 게임 실행 · 게임에서 **T** = 낮/밤 전환
